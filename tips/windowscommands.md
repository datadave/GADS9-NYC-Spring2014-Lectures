# Common Windows Commands

**Note**: These commands assume you are using the vanilla Windows command prompt.

**Note**: Tab completion is your friend.

### Where am I?
    cd

### What's in here?
    dir

### Navigation
    cd <directory>

#### Go up one directory
    cd ..

#### Go up two directories
    cd ..\..\

### Create a directory
    mkdir <directory>

### Scrolling through a file
    more <filename>

### Delete a file
    del <filename>
